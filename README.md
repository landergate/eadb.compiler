<p>db.exe | item database loader</p>
<p>usage</p>
<p>db.exe  -p    &lt;server-path&gt;  -o      &lt;output-path&gt;  -m    &lt;server-type&gt;</p>
<p>db.exe --path &lt;server-path&gt; --output &lt;output-path&gt; --mode &lt;server-type&gt;</p>
<p>&lt;server-path&gt; is the path to the root directory of the server</p>
<p>&lt;output-path&gt; is the path to the directory to hold the database</p>
<p>&lt;server-type&gt; can be "eathena", "rathena", "hercules", "resource"</p>

<p>example</p>
<p>db.exe -p G:\\git\\eathena            -o G:\\db -m eathena</p>
<p>db.exe -p G:\\git\\rathena            -o ..     -m rathena</p>
<p>db.exe -p G:\\git\\hercules           -o .      -m hercules</p>
<p>db.exe -p G:\\git\\eadb.compiler\\res -o G:\\db -m resources</p>
